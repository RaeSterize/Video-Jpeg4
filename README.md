# Video-Jpeg4

<img width="256" height="256" alt="Jpeg4 Icon Page" src="https://github.com/user-attachments/assets/59fd61cf-bd2f-4401-af20-43a23c92d009" />


Ruin your videos quality like a Jpeg!
You can change the framerate & the bitrate for Video & Audio!



## Planned Features
- Allow for Render Iterations. The more you render a single file, the more compressed it gets!
- Downscale resolution (0.75, 0.5, 0.25, etc.)
